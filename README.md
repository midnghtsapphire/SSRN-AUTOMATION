# SSRN Paper Automation System

Automated system for generating, formatting, and publishing academic white papers to SSRN.

## Overview

This system automates the complete workflow for creating professional academic papers:
- Generates papers with authentic human voice
- Enforces strict naming conventions
- Removes all AI watermarks and artifacts
- Creates properly formatted PDFs
- Validates quality and compliance
- Uploads to Google Drive
- Backs up to GitHub
- Extracts metadata and tracks submissions

## Naming Convention

**MANDATORY**: All files MUST follow this format:

```
Walter_Evans_[ShortTitle]_[YYYYMMDD].pdf
```

Examples:
- `Walter_Evans_Quantum_Cognition_20260123.pdf`
- `Walter_Evans_Market_Efficiency_20260124.pdf`

## Quality Requirements

### Content Standards
- ✅ Written in authentic human academic voice
- ✅ No AI watermarks or generation artifacts
- ✅ No author name in body text (only in header)
- ✅ Abstract ≤200 words
- ✅ All required sections present
- ✅ Contra-suggestive titles (opposing concepts)

### Forbidden Terms
The system automatically detects and prevents:
- manus, ai-generated, automated
- machine learning model, language model
- gpt, chatgpt, claude
- "generated by", "ai assistant"
- Any AI-related watermarks

### Metadata Requirements
- Author, title, description, date
- SEO-optimized keywords
- JEL classification codes
- eJournal suggestions
- Clean metadata only (no system info)

## Usage

### Quick Start

Run the complete automation workflow:

```bash
python3 run_automation.py "your paper topic here"
```

Example:
```bash
python3 run_automation.py "quantum cognition in behavioral finance"
```

### Individual Scripts

#### 1. Generate Paper
```bash
python3 scripts/generate_paper.py "your topic"
```

#### 2. Quality Check
```bash
python3 scripts/quality_checker.py path/to/paper.pdf path/to/text.txt
```

#### 3. Extract Metadata
```bash
python3 scripts/extract_metadata.py path/to/paper_data.json
```

#### 4. Upload & Backup
```bash
python3 scripts/upload_and_backup.py path/to/paper.pdf path/to/metadata.json
```

## Workflow Steps

1. **Generate Paper Content**
   - Creates contra-suggestive title
   - Generates subtitle and abstract
   - Writes all sections in human voice
   - Removes AI artifacts

2. **Create PDF**
   - Applies professional formatting
   - Uses academic template
   - Ensures proper spacing and layout

3. **Quality Check**
   - Validates filename convention
   - Checks for forbidden terms
   - Verifies required sections
   - Confirms human writing style

4. **Extract Metadata**
   - Saves paper metadata
   - Updates CSV tracking log
   - Generates submission info
   - Suggests eJournals

5. **Upload to Google Drive**
   - Uploads to folder ID: 1qVmicFsMOLngIoRIAkeVZNreVblXbLic
   - Generates shareable link
   - Verifies successful upload

6. **Backup to GitHub**
   - Copies to SSRN_Whitepapers repository
   - Commits with metadata
   - Pushes to remote
   - Returns commit URL
